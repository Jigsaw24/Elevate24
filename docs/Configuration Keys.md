# Configuration Keys for Elevate24

## `License Configuration`

### License Keys

Required for Elevate24 Premium Customers only. These keys are required to get access to premium functionality. The license is found with the Elevate24 Portal via Jigsaw24. 

```xml
<key>LicenseKey</key>
<string>123456789</string>
```
```xml
<key>LicenseAPIKey</key>
<string>123456789</string>
```
